# Assembly Hello World 

this program simple write 'Hello, World!' in terminal

## Program execute
```bash
./hello_world
Hello, World!
```

## Features
- Architecture - x64-86
- Assembly dialect - NetWide Assembly
- Test platform - Linux Arch

## Quick start
You can paste this in you shell:
```bash
# clone
git clone https://github.com/mental0-main/assembly_hello-world.git

# dir
cd assembly_hello-world

# run
make clean && make run
```
